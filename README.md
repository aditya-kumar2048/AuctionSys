# RIT IPL Auction Simulation 2026 🏏

**Build Smart. Bid Bold. Win Big.**

This project simulates a high-intensity IPL-style auction system where teams compete in real-time to build the strongest squad within a fixed budget.

## 🚀 Features

* Live player bidding system
* Budget management
* Player categorization (Batsmen, Bowlers, All-rounders, Wicketkeepers)
* Real-time auction logic
* Strategy-based team building

## 🧠 Objective

Create a balanced team by bidding strategically while staying within budget constraints.

## ⚙️ Tech Stack

* Frontend: React + Vite + Tailwind CSS
* Backend: Node.js + Express.js
* Database: MongoDB

## 👥 Team Structure

Each team consists of 8 members with defined roles like:

* Mentor
* CEO
* Manager
* Analyst
* Developer roles

## 📌 Rules

* No overspending beyond budget
* No bid withdrawal
* Time-limited bidding
* Fair play required

## 🔗 How to Run

1. Start backend:

   ```bash
   cd backend
   npm install
   npm run dev
   ```
2. Start frontend:

   ```bash
   cd frontend
   npm install
   npm run dev
   ```
